# BMP_Compress
`二值BMP图像按位压缩的C语言实现`  
本算法主要实现对二值bmp图像的压缩   
由于二值bmp图像一个字节表示一个像素  
而且像素数据要么为0x00、要么为0x01  
即一个字节仅用到了最低位，这样便浪费了高七位   
由此，本算法将每八个字节缩位为一个字节，以达到压缩目的  
解压缩时，只需将压缩数据的每个字节按位提取为八个字节即可
